Adds verification meta tag required for claiming the website at Alexa. 

== Description ==

Adding Alexa verification code to your WordPress blog has never been easier. Simply copy and paste your verification meta tag and that's it!

Why claim your website at Alexa? 

* Analyze your website's performance;
* Update website titles and descriptions;
* Respond to reviews;
* ...and more.

For more information visit: [Claim Alexa](http://www.alexa.com/siteowners/claim).

== Installation ==

1. Upload `claim-alexa` directory to the `/wp-content/plugins/` directory or use plugin search - Admin > Plugins > Add new > Search for 'Claim Alexa'.
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Copy verification meta-tag from Alexa and paste it in the settings page (Admin > Settings > Claim Alexa).
4. Click 'Save' and it's done!

Need help? Contact me if you have any questions regarding [how to claim your website at Alexa](http://onlineads.lt/).
